What git command creates a branch?
Answer: git checkout -b

What is the difference between a fast-forward and recursive merge?
Answer: with fast forward merge you have a straight line of history and with
the recursive merge, it is of multiple lines.

What git command changes to another branch?
Answer: git checkout

What git command deletes a branch?
git branch -D

How do merge conflicts happen?
It occurs when your current branch and the branch you want to merge have diverged.
Meaning you have commits in your current branch wich are not in the other one.
